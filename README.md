# sync-scripts

Simple bash scripts for synchronizing local folders with a home server using `rsync` over SSH.

The scripts are designed to be:
- simple and readable
- safe (with dry-run before real sync)
- easy to modify for personal needs

---

## sync-scripts (по-русски)

Простые bash-скрипты для синхронизации локальных папок с домашним сервером с помощью `rsync` по SSH.

Скрипты сделаны с упором на:
- простоту и понятность
- безопасность (сначала dry-run)
- лёгкую адаптацию под личные задачи
